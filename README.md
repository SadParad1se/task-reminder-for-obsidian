# Task Reminder for Obsidian

Android app that reads TaskNotes tasks from selected Obsidian vaults and reminds them to you based on their scheduled and due dates.

The app is read-only. It never writes back to the Obsidian vault.  
Your data never leave your device.  

This app is not affiliated with, endorsed by, or sponsored by Obsidian or the TaskNotes plugin.

## Requirements

This app requires [Obsidian](https://obsidian.md) with the [TaskNotes](https://tasknotes.dev/) and [Advanced URI](https://publish.obsidian.md/advanced-uri-doc/Home) plugins installed.

## Features

- Multiple vaults support
- Reminders for TaskNotes tasks
  - scheduled date/time
  - due date/time
- Takes settings from the TaskNotes (status, priority)
- Simple task filtering
- Open Obsidian from notification
